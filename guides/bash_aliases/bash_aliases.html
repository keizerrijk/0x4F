Bash aliasing

Author: 0x4F


Bash aliases are commands that can be placed within your .bashrc or other bash configuration file such as /etc/bashrc; that can function as a shortcut to other more verbose commands/group of commands.
Alias syntax may look somewhat verbose to the inexperienced users of UNIX/Unix-like operating systems, but after you write 1 or 2 it becomes simple.

The syntax goes as such: alias [name_of_alias]="[command]"
For example I use the following alias to make sure that when I run ls it runs ls -G and displays the directories with colours.
The correct alias for this would be defined as follows: alias ls="ls -G"

Details:

You can list all aliases with the command alias -p
